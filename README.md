# 🎬 CineScope — Movie Discovery App

A modern movie discovery web application built with React, TypeScript, and TMDB API.

## 🚀 Features

- Trending movies with pagination (Load More)
- Search with debounce
- Movie details page
- Trailer integration
- Favorites system
- Loading skeleton
- Error handling with retry

## 🛠️ Tech Stack

* React + TypeScript
* Vite
* Tailwind CSS
* React Router
* TMDB API
