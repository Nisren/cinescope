# 🎬 CineScope — Movie Discovery App

A modern movie discovery web application built with React, TypeScript, and TMDB API.

## 🚀 Features

* 🔥 Trending movies
* 🔍 Search with debounce
* 🎬 Movie details page
* 🎥 Trailer integration (YouTube)
* ❤️ Favorites system (localStorage)
* ⚡ Fast and responsive UI
* 🎨 Modern UI with Tailwind CSS

## 🛠️ Tech Stack

* React + TypeScript
* Vite
* Tailwind CSS
* React Router
* TMDB API
